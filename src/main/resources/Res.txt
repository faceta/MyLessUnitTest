Spring — это самый популярный фреймворком для работы с Java.  В этом курсе вы сможете получить базовые знания о Spring Framework. Курс охватывает следующие темы:

Spring Core: IoC (Inversion Of Control) и DI (Dependency Injection)
Spring AOP (Аспектно-ориентированное программирование)
Spring MVC (Model-View-Controller)
Spring DATA (JPA)
Spring DATA (JDBC)
Spring Boot

Spring Core — ядро фреймворка, на котором держатся все остальные модули. В этом модуле вы познакомитесь с понятиями инверсия управления (Inversion Of Control) и внедрение зависимостей (Dependency Injection).

1. Прочитайте статью "Inversion of Control Containers and the Dependency Injection pattern" от Martin Fowler: 

https://martinfowler.com/articles/injection.html

2. Изучите официальную документацию Spring о IoC Container: 

https://docs.spring.io/spring-framework/docs/current/reference/html/core.html#beans-introduction

3. Прочитайте статью «Spring Dependency Injection»:

https://www.baeldung.com/spring-dependency-injection

4. Посмотрите видеоролик «Dependency Injection»: 

https://www.youtube.com/watch?app=desktop&v=IKD2-MAkXyQ

5. Прочитайте статью «Understanding the @Autowired Annotation in Spring»:

https://www.baeldung.com/spring-autowire

6. Прочитайте статью «Annotation-based Container Configuration»:

https://docs.spring.io/spring-framework/reference/core/beans/annotation-config.html

7. Пройдите урок «Введение в Spring IoC контейнер»:

https://spring-projects.ru/guides/lessons/lesson-2/

1. Прочитайте статью «Introduction to Spring AOP»:

https://www.youtube.com/watch?v=QdyLsX0nG30&list=PLE37064DE302862F8

В этом модуле вы познакомитесь с приложением Spring MVC, узнаете, как начать и настроить свой собственный проект.
